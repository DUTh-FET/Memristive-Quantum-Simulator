**Memristive Quantum Simulator** is a MATLAB-based repository focused on the **design and simulation of quantum circuits using quantum gates**, with a hardware-inspired mapping onto **memristive crossbar architectures**.

The framework provides a fully software-level environment where users can construct quantum algorithms, simulate their execution, and visualize both the **computational results** and the **crossbar programming process** required to implement each quantum gate.

This project explores the intersection of **quantum computing, in-memory computing, and memristive system modeling**, offering insight into how quantum operations can be executed on emerging hardware architectures.

---

## Structure

The repository is organized into the following modules:

- **Crossbar_Generation**  
  Generates crossbar configurations corresponding to quantum gate mappings.

- **Crossbar_Computing**  
  Simulates quantum computations via matrix-vector multiplication.

- **Crossbar_Analysis_Current**  
  Provides analysis tools for inspecting current flow and validating system behavior.

- **GUI_Application**  
  MATLAB App Designer interface for circuit design, simulation, and visualization.

---

## Prerequisites

- MATLAB (R2021a or newer recommended)
- MATLAB App Designer
- Standard MATLAB toolboxes
